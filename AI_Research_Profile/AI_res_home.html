Physics-Informed Machine Learning for Phase Retrieval in Nonlinear Wave Propagation

Phase retrieval is a fundamental inverse problem in nonlinear optics, where detectors measure only intensity, and the phase of a wave remains unobserved. Traditional methods often fail in nonlinear media due to strong coupling between phase and intensity. This project addresses nonlinear phase retrieval using a physics-informed machine learning framework that integrates deep learning with NLSE-based physical constraints to improve prediction accuracy and robustness.

Methodology & Approach
• Generated synthetic datasets using the split-step Fourier method, simulating intensity profiles at multiple propagation distances.
• Designed and trained a multilayer perceptron to predict the phase at a secondary plane from two intensity measurements.
• Incorporated a physics-based regularizer derived from the 1D nonlinear Schrödinger equation to enforce consistency with physical laws.
• Applied Monte Carlo dropout during inference to quantify predictive uncertainty and identify ambiguous regions.

Results & Observations
• Achieved near-perfect intensity reconstruction and low phase errors for in-distribution data.
• Demonstrated robustness to out-of-distribution inputs, with meaningful uncertainty estimates correlating with prediction ambiguity.
• Outperformed classical Gerchberg–Saxton algorithm in both accuracy and stability, showcasing the advantage of combining data-driven learning with physical priors.

Significance
• Provides a framework for solving ill-posed inverse problems in nonlinear wave propagation.
• Highlights the potential of physics-informed machine learning for uncertainty-aware modeling in optical systems.

Model Research & Performance

Target Engineering & Transformation
• Initiated experiments using target transformation techniques with CatBoost to stabilize predictions.
• Observed improved mid-range accuracy but consistent failures at extreme peak values.
• Analyzed distribution skewness to understand instability at high-magnitude targets.
• Applied logarithmic transformation to the target variable to reduce variance.
• Evaluated transformed targets for suitability across different predictive models.
Gradient Boosting Model Experiments
• Transitioned to XGBoost using log-transformed targets to improve peak handling.
• Tuned tree depth, learning rate, and regularization parameters for financial data behavior.
• Achieved smoother predictions across volatile regions with reduced overfitting.
• Identified limitations in capturing abrupt market spikes.
• Compared CatBoost and XGBoost performance using visual error plots.
Autoencoder-Based Feature Learning
• Designed Autoencoder architectures to learn latent representations of asset dynamics.
• Integrated Autoencoder outputs with CatBoost for enhanced feature representation.
• Observed strong performance on higher peaks with reduced noise sensitivity.
• Noted consistent underperformance on lower-end value ranges.
• Evaluated Autoencoder + XGBoost pipelines, which failed to converge reliably.
Deep Learning Architecture Design
• Designed a custom ResNet sandwich architecture to capture hierarchical temporal dependencies in asset time-series data.
